# Assets

This directory contains the visual resources used throughout the **Vereda Inteligente** project.

The assets support the documentation, user interface, and scientific communication of the platform.

## Directory Structure

```text
assets/

├── banners/
│   └── Project banners and repository headers
│
├── screenshots/
│   └── Platform screenshots and user interface images
│
├── diagrams/
│   └── System architecture, workflows, and technical diagrams
│
└── logos/
    └── GeoSpatial Intelligence Lab branding resources
```

## Contents

### Banners

Repository banners and cover images used in the project's documentation.

### Screenshots

Images of the platform interface, dashboards, modules, and operational workflows.

### Diagrams

Technical illustrations describing the platform architecture, data flow, system components, and deployment workflow.

### Logos

Official branding resources of **GeoSpatial Intelligence Lab**, including logos and institutional identity elements.

## Purpose

The resources contained in this directory are intended to improve the documentation, reproducibility, and visual communication of the project while maintaining a consistent branding strategy across the GeoSpatial Intelligence Lab ecosystem.
